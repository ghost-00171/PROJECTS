# Mini Linux Shell with Local AI Integration

## Overview

**Mini Linux Shell with Local AI Integration** is an Operating Systems Lab project developed in **C** that implements a lightweight Linux shell with built-in command support, process management, command history, piping, and local AI integration using **Ollama**. The project demonstrates key operating system concepts such as process creation, system calls, command parsing, and user interaction in a Linux environment.

---

## Project Objectives

The primary objective of this project is to build a custom Linux shell that provides an interactive command-line interface while enhancing the user experience through AI-assisted command explanations. The project focuses on understanding the internal working of operating systems and Linux process management through practical implementation.

---

## Features

- Custom shell interface developed in C
- Built-in Linux commands
- Command history support
- Process creation and management
- Command piping
- Local AI integration using Ollama
- Interactive command execution
- Cybersecurity-related command assistance
- Linux environment testing

---

## Built-in Commands

The shell currently supports several built-in commands, including:

- `cd`
- `pwd`
- `history`
- `clear`
- `help`
- `exit`

Additional Linux commands can also be executed through the shell.

---

## AI Integration

The project integrates **Ollama** as a local AI assistant to provide:

- Linux command guidance
- Cybersecurity-related queries
- General Linux assistance
- Offline AI support without requiring cloud services
